# Movie Catalog Web App

This is a full-stack CRUD web app built with FastAPI, SQLAlchemy, Jinja2, and PostgreSQL.

## Features
- Add, edit, delete, and list movies
- PostgreSQL database
- Styled HTML templates using Jinja2

## How to run
1. Activate virtual environment: venv\Scripts\activate
2. Run the app: uvicorn main:app --reload
3. Open in browser: http://127.0.0.1:8000/
